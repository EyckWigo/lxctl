lxctl
=====

perl program to control lxc-based containers in vzctl-like way.

Contributing
------------

http://www.lxc.tl/bugtrack/

Installation
-----------

    make DESTDIR="/usr/local/" install


Usage
-----

    $ lxctl --help
or
    $ lxctl --man
